# Polymorphisme

Cours suivi à la base sur OC à l'adresse suivante :
https://openclassrooms.com/fr/courses/1894236-programmez-avec-le-langage-c/1898632-mettez-en-oeuvre-le-polymorphisme

J'y ai rajouté ce qui a été demandé par exemple le garage et la gestion de l'ajout / suppression de véhicules.
Les pointeurs restent la partie la plus délicate pour la suppression de la mémoire allouée dynamiquement selon moi.
Il faudrait jetter un oeil à la méthode removeVehicle().

Ne m'en voulez pas d'avoir mis les classes dans le même header car c'est uniquement à but de compréhension !

Vous trouverez également le fichier tasks.json contenant les paramètres de build.
Vous aurez besoin de g++ afin de compiler sous VS Code.
